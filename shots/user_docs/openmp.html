OpenMP Reference

1. Description

OpenMP (Open Multi-Processing) is an API that supports multi-platform shared-memory parallel programming in C, C++, and Fortran. It provides a simple and flexible interface for developing parallel applications by using compiler directives, library routines, and environment variables.

2. Installation

Most modern compilers include OpenMP support by default. To enable OpenMP:

  • GCC (C/C++): Compile with -fopenmp
  • Clang (C/C++): Compile with -fopenmp and link against libomp
  • Intel Compiler (icc/ifort): Compile with -qopenmp

3. Usage

Use compiler directives (pragmas) to parallelize loops and regions:

#pragma omp parallel for
for (int i = 0; i < n; ++i) {
    // parallel work
}

Key directives and routines:

  • #pragma omp parallel – Creates a parallel region
  • #pragma omp for – Distributes loop iterations across threads
  • omp_get_thread_num() – Returns the thread ID
  • omp_get_num_threads() – Returns the number of threads
