Python Data Analysis with Pandas

Description

Pandas is a powerful and flexible open-source data analysis and manipulation library for Python. It provides data structures like Series and DataFrames that allow for efficient handling of structured data. Pandas is particularly useful in high-performance computing (HPC) environments, where it can handle large datasets and perform complex data operations efficiently. It integrates well with other libraries such as NumPy and Matplotlib, making it a popular choice for data analysis tasks in various scientific and academic fields.

Installation

To use Pandas on the Faculty Arts and Sciences Research Computing (FASRC) cluster, you need to load the appropriate Python module. You can choose between the following versions:

  • For Python 3.12 with Anaconda distribution: module load python/3.12.8-fasrc01
  • For Python 3.10 with Anaconda distribution: module load python/3.10.13-fasrc01
  • For Intel distribution of Python: module load intelpython/3.9.16-fasrc01

After loading the module, you can install Pandas using pip:

pip install pandas

Usage

Pandas is used for data manipulation and analysis. Here are some common operations:

  • Creating DataFrames from various data sources (CSV, Excel, SQL, etc.)
  • Data cleaning and preprocessing (handling missing values, filtering, etc.)
  • Data aggregation and grouping
  • Time series analysis
  • Data visualization using integration with Matplotlib

In an HPC environment, it is essential to efficiently manage memory and processing resources. Use chunking for large datasets and consider parallel processing with libraries like Dask when necessary.

Examples

Here are some practical examples of using Pandas for data analysis:

Example 1: Creating a DataFrame

import pandas as pd

data = {'Name': ['Alice', 'Bob', 'Charlie'],
        'Age': [25, 30, 35],
        'City': ['New York', 'Los Angeles', 'Chicago']}

df = pd.DataFrame(data)
print(df)

Example 2: Reading a CSV File

df = pd.read_csv('/n/holyscratch01/data.csv')
print(df.head())

Example 3: Grouping Data

grouped = df.groupby('City').mean()
print(grouped)
